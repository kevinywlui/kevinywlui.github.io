Homework

  • There will be weekly homework due Monday. There is a grader for this course.
  • The grader is allowed to reject any assignment if it is incomprehensible or illegible.
  • Not all problems will be looked at. A few problems will are graded for correctness. A few problems will be graded for completion.
  • The lowest homework score will be dropped.

Quizzes

Outline

  • Roughly, we will go through Chapters 1-6.
  • Topics include vector spaces, bases, linear transformations, matrix operations, determinants, eigen-stuff, inner products.
  • This course is intended to be a second exposure to linear algebra. There will be an emphasis on proofs and rigor. But we will also cover applications and computations (with a computer).
